# 🌍 Planet Satelitní Snímky

Jednoduchá webová aplikace ve Flasku, která pomocí Planet API zobrazí náhodný satelitní snímek Země.

## 🔧 Požadavky

- Python 3.7+
- Flask
- requests
- Platný API klíč z https://www.planet.com/account/#/apikeys
